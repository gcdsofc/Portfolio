## 📌 Sobre

Portfólio pessoal desenvolvido **do zero**, sem frameworks ou bibliotecas externas.

Faz parte do **Degrau 2** do meu cronograma de projetos práticos — onde o foco é dominar os fundamentos reais do desenvolvimento web: estrutura HTML, estilização CSS e interatividade com JavaScript puro, finalizando com deploy real via GitHub Pages.

---

## ✨ Funcionalidades

- 🔤 Efeito de digitação no título ao carregar a página
- 🔮 Menu com highlight automático conforme a rolagem (`IntersectionObserver`)
- 🃏 Cards de projetos com animação de hover
- 📱 Layout responsivo com Flexbox e CSS Grid
- 🌐 Publicado e acessível online via GitHub Pages

## 🚀 Rodar localmente

```bash
git clone https://github.com/gcdsofc/degrau-2-portfolio.git
cd degrau-2-portfolio
```

Abra o `index.html` diretamente no navegador. Sem instalação, sem servidor.
